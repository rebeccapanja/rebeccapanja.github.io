Developer Tool for Consolidating CSS Color Palette

With growing codebases, its easy to end up with tons of CSS colors and manually finding/replacing is no fun, so this tool aims to make life easy. The goal is to extract colors used in your CSS, show you the color palette visually, and give you an easy way to consolidate colors!
EXPERIMENT

Experimental Duration Picker

While there are plenty of datepickers available, there are no web duration pickers for setting your bedtime playlist to stop 30 mins from start or to setup beeps for interval workouts. This project builds a component to do this and is meant to be used as part of a bigger project. The obvious choice for a picker would be a dropdown but this project experiments with keeping the user's focus on their keyboard. This approach eliminates the need for clicking a dropdown, scrolling to find your option and instead auto-formats your input on pressing Enter.
